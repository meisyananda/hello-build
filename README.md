Fitur Utama
📷 Ambil Foto Instan — Foto profil langsung diambil dari kamera depan device lewat expo-camera, tidak perlu upload manual dari galeri.
📍 Deteksi Lokasi Otomatis — Lokasi (kota + koordinat GPS) terdeteksi otomatis lewat expo-location, langsung tampil di kartu profil.
🪐 Tampilan Bertema Planet — UI custom bernuansa luar angkasa (planet, cincin, bintang) yang konsisten di semua layar, dibangun murni dari komponen View tanpa asset gambar tambahan.
ℹ️ Info Versi App — Halaman About menampilkan versi & build number aplikasi secara otomatis via expo-constants, berguna untuk tracking rilis.
